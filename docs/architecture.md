# 系统架构文档

> 本文档详细描述佐佑动漫社 2025 年度总结系统的整体架构设计。

## 总体架构图

```
┌─────────────────────────────────────────────────────────┐
│                    成品页面 (end.html)                     │
│  ┌──────────────────────────────────────────────────────┐│
│  │                    Phase 控制系统                      ││
│  │  body.phase-login → phase-ticket → phase-flight      ││
│  │       → phase-system → phase-dept → phase-report     ││
│  └──────────────────────────────────────────────────────┘│
│                                                          │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌────────────┐ │
│  │ 登录层    │ │ 船票层    │ │ 星系层    │ │ 报告层      │ │
│  │ #login-  │ │ #ticket- │ │ #system- │ │ #report-   │ │
│  │  layer   │ │  layer   │ │  layer   │ │  layer     │ │
│  └──────────┘ └──────────┘ └──────────┘ └────────────┘ │
│                                                          │
│  ┌──────────────────────────────────────────────────────┐│
│  │                  Three.js 3D 引擎                     ││
│  │  星空背景 / 星球系统 / 飞船动画 / 流星雨 / 鼠标拖尾    ││
│  └──────────────────────────────────────────────────────┘│
│                                                          │
│  ┌─────────┐ ┌─────────────┐ ┌────────────────────────┐ │
│  │ 烟花系统 │ │ 安利星图(3D) │ │ Swiper 报告卡片系统     │ │
│  └─────────┘ └─────────────┘ └────────────────────────┘ │
└─────────────────────────────────────────────────────────┘
```

## 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Three.js | r128 | 3D 星系场景、星球交互、飞船动画 |
| GSAP | 3.12.2 | 时间线动画、页面转场 |
| Swiper | 11 | 个人报告卡片滑动 |
| html2canvas | 1.4.1 | 海报截图导出 |
| Animate.css | 4.1.1 | CSS 动画预设 |

## Phase 状态机

```
[登录] ──verifyUser()──→ [船票] ──launchShip()──→ [飞行过渡]
                                                       │
                     ┌──backToSystem()──┐               ↓
                     │                   │          [星系探索]
                     ↓                   │               │
                 [个人报告] ←showReport() ← 6/6完成 ──── │
                                                         │
                              [部门页面] ←showDeptPage()──┘
                                   │
                    backFromDept()──┘
```

## 数据流向

```
离线数据处理 (Python 工具)
      │
      ├─→ DB[]              用户主数据数组（JSON 内嵌）
      ├─→ WEIXIN_ARTICLES[] 微信公众号文章数据
      ├─→ BILIBILI_VIDEOS[] B站视频数据
      └─→ DEPARTMENTS[]     部门星球定义 & 3D 参数

用户输入姓名
      │
      └─→ verifyUser() 匹配 DB[i]
             │
             └─→ currentUser { name, joinTime, depts[], stats, ... }
                       │
                       ├─→ renderSlides(user)        个人报告幻灯片
                       ├─→ showDeptPage(deptId)       部门页面渲染
                       ├─→ renderInfluenceDashboard() 外宣数据看板
                       ├─→ 烟花系统                    年度祝福
                       └─→ 安利星图                    IP推荐展示
```
